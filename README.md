# Postcode Map Project

This project visualizes a list of postcodes on an interactive map using Folium.

## Requirements

- Python 3.x
- Geopy
- Folium
- Pandas
- pdfkit

Install the required packages using:
```bash
pip install -r requirements.txt
```
